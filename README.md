Airbnb의 UI를 참고했습니다. <br/>
프로젝트 배포 주소 : <br/>

##### 주요 기능

- 스터디 등록 (스터디 주제, 지역, 날짜, 미팅 횟수 등 설정)
- 스터디 멤버 관리 (가입 승인, 출결)
- 스터디 일지 (스터디 내용 기록, 평가, 출석률과 평가 통계)
